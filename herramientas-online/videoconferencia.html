Jitsi

  • Se puede instalar en servidores propios
  • Se puede usar con un navegador o con app de móvil: android o Apple iOS
  • También se ofrece como servicio en https://meet.jit.si/
  • No es necesario crear ninguna cuenta, solo crear la sala y compartir el enlace
  • Se puede proteger la sala con una contraseña

Skype

  • Propiedad de Microsoft
  • Uso muy extendido
  • Es necesario descargar la aplicación en Descargar skype
  • Hay que crear una cuenta de usuario

Google Hangouts

  • Propiedad de Google
  • Uso muy extendido
  • Se utiliza a través del navegador, pero hay que instalar complementos
  • También se puede usar con app para móvil/tablet
  • Hay que crear usar la cuenta de Google
